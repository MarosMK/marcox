# Marcox — Automated Excels

A simple website offering free Excel automation tools to download. Each tool comes with a documentation video explaining how to use it.

## Tools

### Combining Spreadsheet
Combine multiple sheets into one. Easily merge data from multiple Excel files into a single unified spreadsheet.

### Mailing Spreadsheet
Send bulk emails via MS Outlook. A simple solution for sending mass emails directly from an Excel sheet.

### Querying Spreadsheet
Run SQL queries directly in Excel. Connect to SQL Server to query or join your Excel data with any database table.
